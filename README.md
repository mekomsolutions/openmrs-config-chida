# OpenMRS config for CHIDA

This is the OpenMRS backend configuration for the [_CHIDA Bahmni distribution_](https://github.com/mekomsolutions/bahmni-distro-chida/). This configuration is designed to be loaded by
- The [Initializer module](https://github.com/mekomsolutions/openmrs-module-initializer), and
- The [Address Hierarchy module](https://github.com/openmrs/openmrs-module-addresshierarchy).
